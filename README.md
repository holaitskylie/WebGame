  html, css, Javascript를 이용하여 두더지잡기를 응용해 만든 게임입니다.  
  두더지 대신에 여우를 괴롭히는 못된 호박을 잡으면 됩니다🎃  
  여우를 잡게되면 스코어를 알려주는 경고창이 뜨면서 게임이 종료됩니다.  
  화면 오른쪽 상단의 버튼으로 BGM을 끄거나 다시 켜실 수 있습니다.
